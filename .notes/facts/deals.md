# Сущность: Сделки (Deals)

## Метаданные
- **Категория**: Основная сущность
- **Обновлено**: 2026-04-14
- **Маршруты**: `/deals`, `/deals/new`, `/deals/:id`

## Обзор
Сделки — это финальные соглашения между клиентами по объектам недвижимости. Представляют собой завершающий этап воронки: запрос → совпадение → показ → сделка.

## Структура данных

```js
{
  id: string,                  // Уникальный ID
  client_id: string,          // Клиент
  property_id: string,        // Объект
  deal_type: string,          // Тип сделки (продажа/аренда)
  price: number,              // Цена сделки
  status: string,             // Статус
  date: ISO8601,              // Дата сделки
  notes: string,              // Заметки
  created_at: ISO8601,
  updated_at: ISO8601
}
```

## UI Компоненты
- `ListPage` — список сделок с фильтрацией
- Страница создания/редактирования

## Действия (Reducer Actions)
- `ADD_DEAL` — создать сделку
- `UPDATE_DEAL` — обновить сделку
- `DELETE_DEAL` — удалить сделку

## Связи с другими сущностями

```
Deal
├── Client (клиент)
├── Property (объект)
├── Showing (предшествующий показ)
└── Match (исходное совпадение)
```

## Воронка продаж

```
Client → Request → Match → Showing → Deal
  ↓         ↓         ↓        ↓       ↓
Клиент   Запрос    Совпадение Показ   Сделка
```
