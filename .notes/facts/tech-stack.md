# 🏗 Факты о проекте Re-Pro

## Основная информация
- **Название**: Re-Pro (realtor-match)
- **Тип**: CRM для риелторов
- **Версия**: 0.0.0
- **Приватный**: да

## Ключевые сущности
- **Properties** — объекты недвижимости
- **Deals** — сделки
- **Matches** — совпадения (клиент ↔ объект)
- **Clients** — клиенты
- **Meetings** — встречи
- **Templates** — шаблоны документов
- **Estimations** — оценки недвижимости

## Локация
- **Город**: Киров (основной)
- **Районы**: Ленинский, Октябрьский, Первомайский, Нововятский
- **Доп. города**: Москва, Санкт-Петербург, Другой

## Типы недвижимости
- Квартира, Дом, Участок, Коммерческая, Комната

## Статусы объектов
- Встреча, АД, В рекламе, Задаток, Сделка, Отказ, Активен, Продан, Резерв, Снят

## Интеграции
- **Supabase**: аутентификация, БД
- **Cloudinary**: загрузка изображений
- **Vercel**: хостинг
