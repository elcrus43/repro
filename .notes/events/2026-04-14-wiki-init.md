# 2026-04-14 — Инициализация Wiki и структуры знаний

## Метаданные
- **Тип**: INIT
- **Дата**: 2026-04-14
- **Участники**: AI assistant + пользователь

## Описание
Создана структура wiki для проекта Re-Pro по паттерну Karpathy + MemPalace.

### Что сделано
1. ✅ Закоммичены изменения: `.qwen/settings.json`, `vite.config.js`
2. ✅ Создана структура `.notes/`:
   - `index.md` — каталог всех страниц
   - `log.md` — журнал действий
   - `schema.md` — правила и соглашения
   - `facts/` — факты о проекте
   - `events/` — события разработки
   - `decisions/` — архитектурные решения (ADRs)
   - `patterns/` — обнаруженные паттерны
   - `issues/` — проблемы и фиксы
   - `_templates/` — шаблоны для новых страниц

### Источники вдохновения
- [Karpathy: LLM Wiki Pattern](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)
- [MemPalace: Local AI Memory System](https://github.com/MemPalace/mempalace)

### Решения
- Интегрированы оба подхода: wiki (Karpathy) + дворец памяти (MemPalace)
- Структура адаптирована под проект React/Vite без полноценного MemPalace (используется как концепция)

## Последствия
- Теперь AI assistant имеет структурированное хранилище знаний о проекте
- Новые знания добавляются инкрементально
- Можно использовать для онбординга новых разработчиков
